# Bob Go Shipping for ERPNext

BobGoShipping connects ERPNext Shipment workflows to Bob Go so teams can rate, book, and track courier shipments without leaving ERPNext.

## Key features

- Fetch available Bob Go courier services from ERPNext Shipments.
- Select a courier service and create the shipment in Bob Go.
- Store shipment identifiers, carrier details, costs, tracking references, and tracking status on ERPNext Shipments.
- Attach printable PDF shipping labels to Shipments.
- Sync tracking updates from Bob Go to ERPNext Shipments and linked Delivery Notes.
- Use a single integration to work with both production and sandbox Bob Go APIs.
